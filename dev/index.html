Get a connection (Socket)

Cuando haya una conección al servidor mediante sockets, el servidor pedira al dispositivo que diga de que casilla es. Para ellos, medianto un on el dispositivo tendrá que realizar un emmit con la casilla que le corresponde a ese dispositivo, por ejemplo (El ejemplo en JS): 
socket.on('whoAreYou', function(){
	socket.emit('IAm', {casilla:NUMERO_CASILLA})
})

Sent evidence (Socket)

El envia de evidencia será divido en 2 secciones, una para guardar la imagen el servidor (La cual regresa el nombre con el que se guardó en el servidor) y la otra para guardad en base de datos y notificar a la plataforma.
La evidencia mediante el socket debe tener la siguiente estructura: 
evidencia = {
	longitude:-100.000000,
	latitude:20.000000,
	fotografia:"NOMBRE_FOTOGRAFÍA",
	descripcion:"DESCRIPCION_QUE_EL_USUARIO_PUSO_EN_LA_APP_AL_CARGAR_LA_EVIDENCIA",
	usuario_id:1,
	tipo_evidencia_id:1
}
NOTA: Los valores de de tipo_evidencia_id son; 2 - acta inicial, 3 - reporte, 4 - comida, 5 - acta final
NOTA: para la comida no es necesario subir una fotografía y la descripcion puede ir pre-escrita
El tag para el socket es: 
socket.emit('upEvidence', evidendia)
La respuesta del servidor será: 
socket.on('responseEvidence', function(data){
	//Tu código va aquí
})
Donde data es un JSon con la estructura: 
{
	status:true/false,
	message:"MENSAJE DE RESPUESTA"
}
